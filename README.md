# 👾 Cuenta Experimental de Xiali

¡Hola! Esta es mi cuenta alternativa, un espacio para experimentar, jugar con ideas y explorar nuevas tecnologías sin presión.
Aquí no hay perfección, solo curiosidad y aprendizaje constante 💡

## 🧠 ¿Qué hago aquí?
- Subo prototipos, ideas incompletas, pruebas técnicas o conceptos que estoy explorando
- Experimento con distintas tecnologías para aprender haciendo

## ⚙️ Tecnologías que uso
- **Frontend:** HTML, CSS, JavaScript, TypeScript, Vue, Tailwind, Astro
- **Backend:** Node.js, Express, Python
- **Bases de datos:** PostgreSQL, MongoDB, Prisma
- **Otros intereses:** IA, NLP, aprendizaje profundo, diseño web
